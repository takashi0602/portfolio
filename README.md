# Takashi's portfolio

[https://takashimelon.vercel.app/](https://takashimelon.vercel.app/)

## Setup

```bash
$ git clone https://github.com/takashi0602/portfolio.git

$ cd portfolio

$ npm install

$ npm run dev
```

## Scripts

```bash
$ npm run dev

$ npm run build

$ npm run start

$ npm run check:eslint

$ npm run check:prettier

$ npm run fix:eslint

$ npm run fix:prettier

$ npm run lint-staged
```
